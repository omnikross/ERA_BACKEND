Запуск:

1.pip install flask
2.В середовищі розробки(в терміналі) вводимо команду python -c "import flask; print(flask.__version__)"     - перевірка на роботоздатність
3.set FLASK_APP    -вказуємо з якою прогою працюватимемо
4.set FLASK_ENV=development    -вказуємо режим розробки
5.flask run     -запуск сервера
Сервер запущений
